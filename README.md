# Cloud Project

Application React avec structure standard.

## Structure
```
src/
├── components/     # Composants réutilisables (Navbar, etc.)
├── pages/          # Pages de l'application
├── utils/          # Fonctions utilitaires et API
├── App.js          # Composant principal avec routing
└── index.js        # Point d'entrée
```

## Installation
```bash
npm install
npm start
```

## Pages disponibles
- `/` - Accueil
- `/login` - Connexion
- `/register` - Inscription
- `/dashboard` - Dashboard
